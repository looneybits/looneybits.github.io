Detailed Description

This class implements the Ackermann Steering Geometry. The principal inputs to calculate the Ackermann angles are: 1.-wheelTrack in meters. 2.-wheelBase in meters. 3.-ackermannRadius in meters.(The desired minimum turning radius)

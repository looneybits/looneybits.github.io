UnityEngine.UI.VariableGridLayoutGroup Class Reference

VariableCridLayoutGroup is used for building the user interface of this game. I've modified this code a little for having the chance of using a relative cell size. This class is used in combination with VariableGridCell, usually takes the parent role, in other words, under the UI gameObject you will find a child gameObject with VariableGridCell component attached.
